# Task

title: Add practical workflow guidance for new stories and bugs

## Goal

Document the real execution flow for a new user story or bug and provide a concrete Node CRUD example as reference.

## Context

The template already includes generic workflows, but users need a more operational guide for how to start and complete a work item. A concrete example makes the process much easier to adopt in real projects.

## Requirements

- Add a practical workflow for executing a work item
- Add a concrete Node example for a basic User CRUD
- Expose the workflow and example in repository documentation and generated project context

## Acceptance Criteria

- Generated projects include a practical workflow for starting and finishing stories or bugs
- Generated projects include a practical Node CRUD example
- Documentation points users to the new workflow and example
