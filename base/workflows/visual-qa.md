# Visual QA Workflow

## Purpose

Verify that UI implementations match design specifications and behave correctly across devices.

---

## Triggers

- frontend changes
- UI/UX updates
- design verification
- visual regression testing

---

## Workflow

### Layout Check

Verify:

- page structure
- spacing
- alignment
- layout consistency

---

### Component Check

Verify:

- component rendering
- interactive states
- animations and transitions

---

### Responsive Check

Test:

- mobile layout
- tablet layout
- desktop layout

---

### Interaction Check

Verify:

- hover states
- focus states
- form interactions
- navigation flows

---

### Final Verification

Perform:

- visual comparison with design specifications
- accessibility checks
- usability validation
