# .NET Basic Stack

Minimal ASP.NET Core Web API with no database or external dependencies.

## Structure

- `Program.cs` – minimal API with health endpoint
- `Api.csproj` – Web SDK only
- `appsettings.json` – basic logging config

## Commands

- `dotnet run` – run the API
- `dotnet build` – build

## Extending

Add controllers, services, or packages as the project grows. No migrations or database setup required.
