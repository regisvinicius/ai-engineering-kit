# Debug Issue

Investigate and resolve a problem in the codebase.

Steps:

1. Understand the reported issue
2. Reproduce the problem if possible
3. Identify the root cause
4. Propose a minimal fix
5. Implement the fix
6. Validate that the issue is resolved
7. Ensure no regressions are introduced

Focus on:

- root cause analysis
- minimal changes
- stability of the system

Output:

- description of the issue
- root cause
- implemented fix
