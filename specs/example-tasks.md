# Example Tasks

## Goal

Add example task files that demonstrate how this repository tracks work through backlog, in-progress, and done.

## Context

The repository now includes a task workflow and a reusable task template, but it does not yet show what real task files should look like. Example tasks make the workflow easier to understand and give contributors a concrete starting point.

## Requirements

- Add at least one example task in `/tasks/backlog`
- Add at least one example task in `/tasks/in-progress`
- Add at least one example task in `/tasks/done`
- Ensure each example follows `/tasks/TASK_TEMPLATE.md`
- Use examples that fit this repository's purpose as an AI engineering project template

## Acceptance Criteria

- The task-state directories contain example Markdown task files
- Each example task includes title, goal, context, requirements, and acceptance criteria
- The examples illustrate realistic work for this repository
